LIVE TIMESTAMP
===============

Plugin para data por extenso.


Para incluir em seu projeto:

    "require": {
        "dflourusso/live-timestamp": "0.1.*"
    },
    "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/dflourusso/live-timestamp.git"
        }
    ]



Para usar:

echo getExtenso::getContent($date);
